# INFOMATE v11.1: GLOBAL SEARCH BRIDGE UPGRADE

To fix the "Neural Link Rejected" and "Limited Internal Buffers" issue, please deploy the following code to your Supabase Edge Function `global-search-proxy`.

## 1. Setup API Key
Run this in your terminal to set a Serper.dev API key (free tier available):
`supabase secrets set SEARCH_API_KEY=your_serper_api_key`

## 2. Updated global-search-proxy/index.ts
```typescript
import { serve } from "https://deno.land/std@0.168.0/http/server.ts"

const corsHeaders = {
  'Access-Control-Allow-Origin': '*',
  'Access-Control-Allow-Headers': 'authorization, x-client-info, apikey, content-type',
}

serve(async (req) => {
  if (req.method === 'OPTIONS') return new Response('ok', { headers: corsHeaders })

  try {
    const { query } = await req.json()
    const apiKey = Deno.env.get('SEARCH_API_KEY')

    if (!apiKey) {
      return new Response(
        JSON.stringify({ synthesis: "ERROR: SEARCH_API_KEY missing in Supabase Secrets." }),
        { headers: { ...corsHeaders, 'Content-Type': 'application/json' }, status: 200 }
      )
    }

    // Call Serper.dev for real-time Google results
    const response = await fetch("https://google.serper.dev/search", {
      method: "POST",
      headers: {
        "X-API-KEY": apiKey,
        "Content-Type": "application/json",
      },
      body: JSON.stringify({ q: query }),
    })

    const data = await response.json()

    // Synthesize snippets for the LLM
    const snippets = data.organic?.map((res: any) => res.snippet).join("\n") || "No snippets found."
    const synthesis = `SYNTHESIZED_SEARCH_DATA: ${snippets.substring(0, 1000)}`

    return new Response(
      JSON.stringify({ synthesis }),
      { headers: { ...corsHeaders, 'Content-Type': 'application/json' }, status: 200 }
    )

  } catch (error) {
    return new Response(JSON.stringify({ error: error.message }), {
      headers: { ...corsHeaders, 'Content-Type': 'application/json' },
      status: 400,
    })
  }
})
```

## 3. Deployment
Deploy with: `supabase functions deploy global-search-proxy`
